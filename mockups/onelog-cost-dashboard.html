Kế hoạch triển khai · 2 phase
Phase A · Quick win ~2-3 giờ
Chỉ dùng nguồn ① LiteLLM logs
  • 0 code mới · data đã có trong VL từ ngày đầu deploy
  • Thêm Grafana container vào compose (profile dashboard)
  • Import 4 panel: KPI · cost 30d · per-user · fallback events
  • vmalert rule: fallback rate > 20/day, per-user > $2/day
Đủ dùng khi: chỉ cần "ai xài bao nhiêu tuần này" + trend cost

  • Script infra/scripts/poll-provider-cost.sh
  • Tạo admin key riêng cho OpenAI + Anthropic (chmod 0400)
  • Cron 15 phút · emit JSON syslog · Vector tag provider_cost
  • Panel "Balance card" cho 3 provider · Gemini vẫn estimate
  • vmalert: DeepSeek balance < $5 · OpenAI daily > $3
