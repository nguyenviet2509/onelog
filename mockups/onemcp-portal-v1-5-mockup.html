SEPay webhook 500 — restart worker + check DLQ

runbook published · author tri · v3 · updated 34m ago · 👁️ 42 views

Symptoms

  • SEPay webhook trả 500 rải rác từ 09:00
  • Alertmanager firing sepay_webhook_5xx_ratio > 0.1
  • Worker log ECONNREFUSED redis:6379

Verify

docker ps | grep sepay-worker
docker logs --tail 50 sepay-worker
redis-cli -h redis ping

Mitigation

  1. Restart worker: docker restart sepay-worker
  2. Kiểm tra DLQ: redis-cli LLEN sepay:dlq
  3. Nếu DLQ > 100: replay bằng node scripts/dlq-replay.js

Verification

Sau 5 phút: curl -s https://onelog/api/sepay/health phải trả 200. Metric sepay_webhook_5xx_ratio về < 0.01.

Escalation

Nếu DLQ > 500 hoặc replay fail → SRE oncall (@sre-p1 Slack).
